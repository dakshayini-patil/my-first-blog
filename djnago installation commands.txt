Installing packages with requirements

First create a requirements.txt file inside of the django/ folder, using the code editor that you installed earlier. You do this by opening a new file in the code editor and then saving it as requirements.txt in the django/ folder. 
In your django/requirements.txt file you should add the following text:

django/requirements.txt
Django~=5.2.12

Now, run pip install -r requirements.txt to install Django.
